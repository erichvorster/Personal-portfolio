02. Some things I've built

Below are a few of my personal projects. The source code and some of my other work is available on Github

Cryptocurrency App

A fully responsive cryptocurrency app that includes live coin prices, statistics and news updates built with React using function based components. The app also includes several graphs built the help of ChartJS. I made use of the browsers built in fetch api to pull data from both the Coinranking and Bing News api's. The ui was built with Material UI.

  • React
  • React Router
  • Rapid API
  • ChartJS
  • Material UI

Task Manager

A fully responsive task manager built with React and Tailwind CSS. I made use of React function based components and react hooks to achieve CRUD operations. I also made use of Framer Motion to add a few animations.

  • React
  • Tailwind CSS
  • Framer Motion

Internet Provider Website

A fully responsive internet provider website built with Bootstrap 5 and SASS. I made use of flex box and CSS grid to achieve responsiveness. I also made use of GSAP animation library to include some animation.

  • JavaScript
  • Bootstrap
  • SASS
  • GSAP
